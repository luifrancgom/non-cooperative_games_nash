3  Existence of Equilibrium Points

I have previously published [Proc. N. A. S. 36 (1950) 48-49] [1] a proof of the result below based on Kakutani’s generalized fixed point theorem. The proof given here uses the Brouwer theorem.

The method is to set up a sequence of continuous mappings: \(\symscr{s} \to \symscr{s}'(\symscr{s}, 1); \symscr{s} \to \symscr{s}'(\symscr{s}, 2); \ldots\) whose fixed points have an equilibrium point as a1 limit point. A limit mapping exists, but is discontinuous, and need not have any fixed points.

Theorem 3.1 Every finite game has an equilibrium point.

Proof. Using our standard notation, let \(\symscr{s}\) be an n-tuple of mixed strategies, and \(p_{i\alpha}(\symscr{s})\) the pay-off to player \(i\) if he uses his pure strategy \(\pi_{i\alpha}\) and the others use their respective mixed strategies in \(\symscr{s}\). For each integer \(\lambda\) we define the following continuous functions of \(\symscr{s}\):

\[ \begin{aligned} q_i(\symscr{s}) &= \max_{\alpha} p_{i\alpha}(\symscr{s}), \\ \phi_{i\alpha}(\symscr{s}, \lambda) &= p_{i\alpha}(\symscr{s}) - q_i(\symscr{s}) + \frac{1}{\lambda}, \text{ and} \\ \phi_{i\alpha}^{+}(\symscr{s}, \lambda) &= \max [0, \phi_{i\alpha}(\symscr{s}, \lambda)]. \end{aligned} \]

Now \(\sum_{\alpha} \phi_{i\alpha}^{+}(\symscr{s}, \lambda) \geq \max_{\alpha} \phi_{i\alpha}^{+}(\symscr{s}, \lambda) = \frac{1}{\lambda} > 0\) so that \(c_{i\alpha }'(\symscr{s}, \lambda) = \frac{\phi_{i\alpha}^{+}(\symscr{s}, \lambda)}{\sum_{\beta } \phi_{i\beta}^{+}(\symscr{s}, \lambda)}\) is continuous.

Define \(s'_i(\symscr{s}, \lambda) = \sum_{\alpha } \pi_{i\alpha} c'_{i\alpha}(\symscr{s}, \lambda)\) and \(\symscr{s}'(\symscr{s}, \lambda) = (s_1', s_2', \ldots, s_n')\). Since all the operations have preserved continuity, the mapping \(\symscr{s} \to \symscr{s}'(\symscr{s}, \lambda)\) is continuous; and since the space of n-tuples, \(\symscr{s}\), is a cell, there must be a fixed point for each \(\lambda\). Hence there will be a subsequence \(\symscr{s}_{\mu}\), converging to \(\symscr{s}^{*}\), where \(\symscr{s}_{\mu}\) is fixed under the mapping \(\symscr{s} \to \symscr{s}'(\symscr{s}, \lambda_{(\mu)})\).

Now supposed \(\symscr{s}^{*}\) were not an equilibrium point. Then if \(\symscr{s}^{*} = (s_1^*, \ldots, s_n^*)\) some component \(s_i^*\) must be non-optimal against the others, which means \(s_i^*\) uses some pure strategy \(\pi_{i\alpha}\) which is non-optimal [see 2.4, pg. 4]. This means that \(p_{i\alpha}(\symscr{s}^{*}) < q_i(\symscr{s}^{*})\) which justifies writing \(p_{i\alpha}(\symscr{s}^{*}) - q_i(\symscr{s}^{*}) < -\epsilon\).

From continuity, if \(\mu\) is large enough,

\[ \Big| [p_{i\alpha}(\symscr{s}_{\mu}) - q_i(\symscr{s}_{\mu})] - [p_{i\alpha}(\symscr{s}^{*}) - q_i(\symscr{s}^{*})] \Big| < \frac{\epsilon}{2} \text{ and } \frac{1}{\lambda_{(\mu)}} < \frac{\epsilon}{2}. \]

Adding, \(p_{i\alpha}(\symscr{s}_{\mu}) - q_i(\symscr{s}_{\mu}) + \frac{1}{\lambda_{(\mu)}} < 0\) which is simply \(\phi_{i\alpha}(\symscr{s}_{\mu}, \lambda_{(\mu)}) < 0\), whence \(\phi_{i\alpha}^{+}(\symscr{s}_{\mu}, \lambda_{(\mu)}) = 0\), whence \(c_{i\alpha}'(\symscr{s}_{\mu}, \lambda_{(\mu)}) = 0\). From this last equation we know that \(\pi_{i\alpha}\) is not used in \(\symscr{s}_{\mu}\) since \(\symscr{s}_{\mu}=\sum_{\alpha} \pi_{i\alpha} c_{i\alpha}'(\symscr{s}_{\mu}, \lambda_{(\mu)})\), because \(\symscr{s}_{\mu}\) is a fixed point.

And since \(\symscr{s}_{\mu} \to \symscr{s}^{*}\), \(\pi_{i\alpha}\) is not used in \(\symscr{s}^{*}\), which contradicts our assumption.

Hence \(\symscr{s}^{*}\) is indeed an equilibrium point.
